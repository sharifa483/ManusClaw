Massive Evolution

What's New in v5.0.0?

ManusClaw is no longer just a chatbot wrapper. It has evolved into a true omnichannel AI platform.

12+ Messaging Channels

Agents now operate natively across Telegram, Discord, Slack, WhatsApp, Signal, Teams, Matrix, WebChat, and Email with cross-platform routing.

Voice Wake & Talk Mode

Includes Porcupine wake-word detection (e.g., "Hey Manus") and a continuous talk loop. Command the agent via microphone and get TTS responses.

Live Canvas (A2UI)

Agent-to-UI protocol enabled. Agents can now render real-time UI components, charts, tables, and interactive buttons via WebSockets.

Webhooks & Gmail Pub/Sub

Features HMAC-verified incoming webhooks and Gmail Pub/Sub automation. Agents can handle tasks and auto-replies immediately as emails arrive.

System Intelligence

Multi-Agent Orchestration

A fully independent DAG pipeline operating on a strict PAORR (Plan -> Act -> Observe -> Reflect -> Retry) execution loop.

1. Product Manager 📝

Analyzes task requirements and generates a PRD (Product Requirement Document). Sets clear, measurable goals.

2. Architect 🏛️

Plans the software architecture, defines directory trees and the tech stack. Creates a perfect DAG execution plan.

3. Engineer (Manus) ⚙️

Utilizes the 14-tool arsenal to write code, execute it, and autonomously fix errors without human intervention.

4. QA Role 🔬

Ensures quality assurance by running rigorous tests and validating that everything is production-ready according to the PRD.

Lethal Arsenal

14 Built-in Core Tools

The agent uses these tools to interact with the real world. Soft shapes hiding industrial power.

Persistent Bash

Stateful shell sessions that remain active in memory. Control the OS natively across tool calls.

Isolated PythonExec

Executes code in a fresh, isolated multiprocessing sandbox with a strict 2GB memory limit.

BrowserUse Autonomy

Playwright-driven Chromium automation. Grants the freedom to navigate, click, and extract web data.

EmailTool & Gmail

New addition! The agent can now read, send, and reply in threads autonomously.

CanvasTool & DataViz

Renders live charts, UI components, and interactive elements directly to connected screens.

Cross-Session Memory

SQLite + FTS5 memory integration. Grants the ability to recall past sessions and learn from failures.

Internal Engines

Digital That Feels Real

The framework's solid logic and robust security systems are represented through these realistic elements.

PAORR & Fallback Logic

Tools are auto-penalized on failure. Features seamless failover rotation between LLM providers (OpenAI, Groq, Anthropic).

━ ━ calculated

Identity & Permissions

IdentityGuard stops jailbreaks using 30+ regex patterns. The 3-tier Permission Gate blocks the execution of dangerous commands.

◇ ◇ secured

Audit Ledger & SQLite

WAL-mode SQLite logging. Every step, tool call, stdout, and error trace is permanently recorded. Zero data loss.

Deployment

Run Anywhere You Want

ManusClaw can be deployed in any environment, from heavy backend servers to mobile smartphones.

Standalone CLI

Wake up the AI shell with the native terminal command manusclaw. Slash commands included.

SSH Remote Gateway

New feature! Runs an asyncssh server on port 2222, providing remote access via a restricted shell.

Desktop & Windows Hub

Includes Flet Desktop GUI, macOS Menu Bar, and Windows System Tray companion applications.

3 Sandbox Backends

Supports Docker, SSH remote host, and OpenShell (Linux namespaces) for secure code isolation.

# Installing ManusClaw v5.0.0 is straightforward

git clone https://github.com/ManusClawAI/manusclaw.git
cd manusclaw
chmod +x install.sh && ./install.sh

# Run with all v5.0 features (Voice, Channels, UI)
pip install -e ".[all-plus]"

# Start the Voice Wake Mode
manusclaw voice wake --start --word "hey manus"
